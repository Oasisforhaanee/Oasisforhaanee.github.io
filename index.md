# 안녕하세요 👋

이곳은 **GitHub Pages + Jekyll** 로 만든 나만의 블로그입니다.
오른쪽 상단 메뉴의 **About** 에서 소개를 볼 수 있어요.

## 이 블로그는
- 글은 **마크다운(`.md`)** 으로 작성합니다.
- `_posts` 폴더에 글을 넣으면 아래 **최근 글** 목록에 자동으로 모입니다.
- 파일을 GitHub에 `push` 하면 몇 분 뒤 블로그에 자동 반영됩니다.

## 글을 새로 쓰려면
`_posts` 폴더에 `YYYY-MM-DD-영문슬러그.md` 형식으로 파일을 추가하세요.
자세한 사용법은 저장소의 `README.md` 를 참고하세요.
